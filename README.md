## Features :

### Web server :

- Random reactive billboard
- Information modal
- TV Shows episodes grouped by season
- Full text search
- Search by genre
- Media download
- Media uploader
- Account Menu
- Notifications
- Admin panel
- React video Player
- Vlc streaming tutorial and downloads
